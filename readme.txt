== Description ==
Simple WooCommerce Customize is a simple WordPress plugin which lets user to change the WooCoomerce Product "Add to Cart" button directly from the WordPress Customize.


== Installation ==

= Manual Installation =

1. Upload the entire `/simple-woocommerce-customizer` directory to the `/wp-content/plugins/` directory.
2. Activate Simple WooCommerce Simple Customizer through the 'Plugins' menu in WordPress.
